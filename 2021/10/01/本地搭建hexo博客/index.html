本地搭建hexo博客

发表于 2021-10-01 分类于 hexo博客框架 Valine：

1. 搭建环境

安装 Git https://git-scm.com/
安装 Nodejs https://nodejs.org/en/

配置自定义的全局模块安装位置和缓存位置，新建两个文件夹 node_global 和 node_cache （配置在c盘可能存在权限问题）

然后在cmd管理员权限下执行如下两个命令：
npm config set prefix C:\node_global
npm config set cache C:\node_cache
报错！2022-06-09
npm WARN config global --global, --local are deprecated. Use --location=global instead.
解决方法，打开nodejs安装目录，修改npm.cmd文件，将 prefix -g 修改为 prefix –location=global 即可

以下命令可查看当前全局模块安装位置和缓存位置
npm config get prefix
npm config get cache

执行成功后，在环境变量→系统变量中新建一个变量名为 NODE_PATH，值为：C:\Program Files\nodejs\node_modules

最后编辑用户变量Path，将相应npm的路径修改为：C:\node_global

完成以上在 cmd 执行 npm install webpack -g 安装成功后模块已经在自定义文件夹中了

2. 安装hexo

建立博客文件夹，然后管理员权限下打开 Git Bash 通过cd命令移动到博客文件夹内，输入 npm install -g hexo-cli 命令安装Hexo

安装完成后，输入 hexo init 命令初始化博客
然后输入 hexo g 静态部署
这时网页已经部署完成，输入 hexo s 命令可查看
浏览器输入 http://localhost:4000/ 就可以打开新部署的网页
注：hexo clean && hexo g && hexo s
