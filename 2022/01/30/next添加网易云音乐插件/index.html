添加网易云音乐插件

发表于 2022-01-30 分类于 next更新日志

1. 添加网易云音乐插件

在网易云生成外链播放器
打开blog/themes/next/layout/_macro/sidebar.swig
将外链复制在合适的位置，例如：

<iframe frameborder="no" border="0" marginwidth="0" marginheight="0" width=330 height=86 src="//music.163.com/outchain/player?type=2&id=1837804830&auto=0&height=66"></iframe>

</div>
</aside>
<div id="sidebar-dimmer"></div>
{% endmacro %}

2. 设置全局播放不间断

这里使用pjax插件实现功能
Git Bash进入blog\themes\next
输入 git clone https://github.com/theme-next/theme-next-pjax source/lib/pjax 安装pjax插件
在next配置文件启用pjax插件，如下
pjax: true
