5. 生成密钥

在Git Bash 输入 ssh-keygen -t rsa 生成密钥文件
打开密钥文件 Windows 10位置：C:/Users/自己用户名/.ssh cat id_rsa.pub打开密钥，复制密钥到自己github→Settings→SSH and GPG kesy即可
验证是否成功，可以通过在 Git Bash 中输入 ssh -T git@github.com 进行检验 （第一次需要填yes）
如 Hi xyuanyy! You’ve successfully authenticated, but GitHub does not provide shell access.则绑定成功

6. 设置 user.email 和 user.name

在Git Bash命令下输入
git config –global user.email “you@example.com“
git config –global user.name “Your Name”

7. 部署到github

回到 Git Bash 输入部署命令
hexo clean && hexo g && hexo d
（可能因为网络原因报错，多试几次）
